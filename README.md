# 1С Ассистент — Telegram-бот для онбординга сотрудников

Телеграм-бот, который помогает новым сотрудникам быстрее освоить работу в 1С (ERP). Вместо того чтобы отвлекать руководителя или листать длинные инструкции, сотрудник просто пишет вопрос в чат — и получает конкретный ответ.

---

## Зачем это нужно

Онбординг в 1С — это стресс. Новый сотрудник не помнит алгоритмы, боится ошибиться, постоянно дёргает коллег. Руководитель теряет время на одни и те же вопросы.

Этот бот решает проблему просто: база знаний компании всегда под рукой, в Telegram, в любой момент.

**Что получает сотрудник:**
- Быстрый ответ на вопрос в свободной формулировке
- Пошаговые инструкции по работе в 1С
- Уверенность в сложных ситуациях

**Что получает руководитель:**
- Меньше отвлечений на базовые вопросы
- Новые сотрудники выходят на самостоятельную работу быстрее

---

## Как это работает

Бот использует технологию RAG (Retrieval-Augmented Generation): когда сотрудник задаёт вопрос, бот ищет релевантные фрагменты из базы знаний и передаёт их языковой модели, которая формулирует чёткий ответ.

```
Сотрудник пишет вопрос
        ↓
Бот ищет нужный фрагмент в базе знаний (векторный поиск, ChromaDB)
        ↓
Языковая модель формирует ответ на основе найденного (OpenRouter)
        ↓
Сотрудник получает конкретную инструкцию
```

---

## Технологии

| Компонент | Инструмент |
|---|---|
| Telegram-бот | aiogram 3 |
| Языковая модель | OpenRouter (stepfun/step-3.5-flash:free + fallback-цепочка) |
| Векторная база знаний | ChromaDB |
| RAG-пайплайн | LangChain |
| Эмбеддинги | sentence-transformers (all-MiniLM-L6-v2) |
| LLM-клиент | openai (с кастомным base_url) |
| Язык | Python 3.11+ |

---

## Требования к запуску

- Python 3.11+
- Telegram Bot Token (получить у @BotFather)
- API ключ OpenRouter (бесплатно на openrouter.ai)
- ~300 МБ свободного места на диске

## Установка

```bash
git clone https://github.com/ваш-username/1c-assistant-bot
cd 1c-onboarding-bot
pip install -r requirements.txt
```

Создайте файл `.env`:

```env
TELEGRAM_TOKEN=ваш_токен
OPENROUTER_API_KEY=ваш_ключ
```

Загрузите базу знаний (PDF, Word-файлы с инструкциями по 1С):

```bash
python load_documents.py
```

Запустите бота:

```bash
python main.py
```
